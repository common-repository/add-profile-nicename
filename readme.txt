Make it possible to add nicename in the profile screen.

== Description ==

* You can add a nicename in the profile screen.
* You can change the user name on the contributor archive page to nicename.

= Credits =

This plugin is created by <a href="https://webone-sendai.co.jp/">ホームページ制作ウェブワン仙台</a>.

== Installation ==

1. From the WP admin dashboard, click “Plugins” -> “Add new”.
2. In the browser input box, type “Add Profile Nicename”.
3. Select the “Add Profile Nicename” plugin and click “Install Now”.
4. Activate the plugin.

== Frequently Asked Questions ==

= Is there a character limit? =
You can enter up to 20 characters.
